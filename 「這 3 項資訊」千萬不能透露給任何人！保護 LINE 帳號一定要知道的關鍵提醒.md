近期通訊軟體 LINE 罕見透過蓋板廣告提醒用戶：「保護 LINE 帳號，請勿將帳號設定資料提供給任何人。」點擊後更會導向官方教學文章，教導用戶如何避免帳號遭盜。

這項動作也被外界解讀為，近期詐騙與帳號盜用事件頻傳，LINE 官方不得不主動示警。究竟該如何保護自己的 LINE 帳號不被有心人士盜用？以下整理 LINE 官方重點提醒，一定要牢記。

【重要提醒】就算是熟人，也絕對不能提供這 3 項 LINE 帳號資訊
你是否曾遇過以下情況？

「朋友突然傳訊息，說需要你幫忙填寫或告知 LINE 認證碼。」

無論對方你是否認識，甚至是非常熟悉的親友，只要對方要求你提供 LINE 帳號相關資訊，都必須立刻提高警覺。
因為對方的社群帳號，很可能早已被盜用，實際與你對話的人，已經不是帳號本人。

請務必牢記，以下 3 項資訊，絕對不能告訴任何人：
LINE 帳號設定的「電話號碼」
LINE 帳號設定的「密碼」
LINE 驗證簡訊中的「認證碼」
:warning: 只要其中任一項外洩，帳號就可能在極短時間內被完整盜走。

【務必留意】LINE 官方「沒有」任何所謂的「輔助認證」機制
LINE 也特別強調：

官方從未提供任何需要同時輸入電話、密碼與簡訊認證碼的「輔助認證」流程。

如果你在不明網站或連結中看到類似以下要求：

要你輸入 LINE 帳號設定的電話
要你輸入 LINE 密碼
要你填寫簡訊收到的認證碼
這 100% 是詐騙頁面。
只要照做，帳號就會立刻被盜，請務必關閉頁面，切勿填寫任何資料。

【教學說明】LINE 帳號若已遭盜用，請立即通報客服
若不幸發現 LINE 帳號疑似遭盜，請不要慌張，並 務必由帳號擁有者本人 儘速向 LINE 客服通報。

LINE 客服在確認帳號遭盜後，將會：

立即停權被盜帳號，防止進一步損害
協助用戶重新建立新帳號
引導轉移原帳號中的付費商品與服務
及早回報，是降低損失的關鍵。

在享受各種便利網路服務的同時，資安風險也隨之而來。
提高警覺、不輕信、不透露關鍵資訊，才是保護自己帳號最重要的第一步。
